# The Youth – Sanity Studio

Studio is gekoppeld aan Sanity-project **dpv7nblq** (dataset: `production`).

## Ontwikkeling

```bash
cd studio
npm install
npm run dev
```

Studio draait op **http://localhost:3333/studio** (basePath `/studio`). Via de website: start ook `web` met `npm run dev` en open **http://localhost:5173/studio**.

**Let op:** Sanity Studio v3 vereist Node.js 20.19+ of 22.12+. Upgrade Node indien nodig:

```bash
node -v   # controleer versie
# Gebruik nvm: nvm install 22 && nvm use 22
```
